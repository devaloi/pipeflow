# Changelog

All notable changes to pipeflow are documented here.

## [0.2.0] - 2026-02-20

### Added
- Parquet output connector
- Pipeline visualization (ASCII DAG)
- GitHub Actions CI pipeline

## [0.1.0] - 2026-02-18

### Added
- Source connectors: CSV, JSON, REST API
- Transform steps: filter, map, enrich, validate, join
- Output connectors: SQLite, CSV, JSON
- Composable pipeline builder with method chaining
- Built-in validation with error row tracking
